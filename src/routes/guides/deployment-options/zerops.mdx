[Zerops](https://zerops.io) is a dev-first cloud platform that can be used to deploy both Static and SSR Solid Node.js App. For additional one to one support, details and features, you can join [Zerops Discord server](https://discord.gg/xxzmJSDKPT) and [visit the Zerops website](https://zerops.io).

Deploy and test Zerops Solid recipes with one click: 
- [Solid Node.js](https://app.zerops.io/recipe/solidjs-nodejs) 
- [Solid Static](https://app.zerops.io/recipe/solidjs-static)
